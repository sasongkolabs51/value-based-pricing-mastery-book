*Before you run the calculations, take a quick reading of how ready you feel. This is a checkpoint, not a test.*

| Statement | 1 (Not at all) | 2 | 3 | 4 | 5 (Absolutely) |
|-----------|----------------|----|----|----|-----------------|
| I can name at least three client engagements where I know—roughly—the financial value at stake for the client. |  |  |  |  |  |
| I understand the difference between outcome significance, risk absorption, and judgment intensity as pricing drivers. |  |  |  |  |  |
| I have completed a susceptibility audit (Chapter 4) and can access the scores for the services I want to price. |  |  |  |  |  |
| I’m mentally prepared to propose a fee that might be 3–10× my old hourly estimate, provided the math supports it. |  |  |  |  |  |
| I’ve reviewed my Disaster Portfolio (Chapter 2) recently and can recall specific outcomes I’ve delivered. |  |  |  |  |  |
| I’m willing to have a conversation with a client about the financial risk they transfer to my firm—even if it’s uncomfortable at first. |  |  |  |  |  |

**Interpretation:**  
- If you have any low scores on statements 1–3, revisit the relevant sections of Chapter 5 and your audit data before diving into the calculator.  
- Low scores on statements 4–6 are normal—this chapter is designed to shift that discomfort. The drills will help.
