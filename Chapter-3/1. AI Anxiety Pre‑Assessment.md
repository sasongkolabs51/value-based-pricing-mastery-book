## 1. AI Anxiety Pre‑Assessment  
*Before you start scanning, take a quick reading of your current fear level. This isn’t a test; it’s a baseline so you can see the shift after you’ve applied the tools.*

| Statement | 1 (Not at all) | 2 | 3 | 4 | 5 (Completely) |
|-----------|---------------|----|----|----|-----------------|
| I feel a knot in my stomach when I see an AI-generated deliverable that resembles my team’s work. |  |  |  |  |  |
| I check Fishbowl or similar forums for “AI and billing” threads more than once a week. |  |  |  |  |  |
| My first reaction to client AI efficiency challenges is defensive—I worry about looking overpriced. |  |  |  |  |  |
| I sometimes avoid thinking about AI’s impact on my practice because it feels overwhelming. |  |  |  |  |  |
| I can name specific parts of my work that are highly automatable, and that scares me. |  |  |  |  |  |
| If a client asked me tomorrow, “Why should I pay your rate when GPT‑6 can do this?” I’d struggle to answer without mentioning effort. |  |  |  |  |  |
| I believe AI will shrink the revenue of my practice area within three years. |  |  |  |  |  |

**Scoring:** Add up your numbers.  
- **7–15:** Your anxiety is manageable and you already have some intuitive separation between process and judgment. The scan will sharpen that.  
- **16–25:** Moderate anxiety. The fear is real but not paralyzing—perfect condition for converting worry into a plan.  
- **26–35:** High anxiety. The Fishbowl panic lives in you. Don’t judge it; this chapter is built exactly for where you are. By the end of the exercises, that knot will have coordinates instead of an echo chamber.

*Return to this after completing the toolkit and note your new score.*
