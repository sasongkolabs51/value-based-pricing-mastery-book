## 2. The Mastery Loop: Quarterly AI Capability & Pricing Review

This is the heartbeat of your adaptation engine. A two-hour session, once per quarter, that replaces anxiety with a single, testable adjustment. Use this template as your standing agenda and log.

### Standing Agenda

| Time | Agenda Item | Notes |
|------|-------------|-------|
| 10 min | **Scan:** What’s changed in AI since last quarter? (new models, competitor moves, client-side automation) | |
| 20 min | **Review signals:** Which outcome-priced engagements showed strain? (realization dips, client pushback, scope creep) | |
| 20 min | **Audit spot-check:** Pick 2-3 service lines. Have any audit scores shifted? (standardization, data dependency, client perception) | |
| 30 min | **Adjustment decision:** What’s the *one* change we’ll test? (VRR calibration, packaging language, retraining script update) | |
| 20 min | **Micro-pilot design:** Outline a one-paragraph hypothesis and a single engagement to test it. | |
| 10 min | **Innovation Reserve allocation:** Review reserve balance and decide how to deploy it this quarter. | |
| 10 min | **Next steps:** Assign actions, set review date for next quarter. | |
