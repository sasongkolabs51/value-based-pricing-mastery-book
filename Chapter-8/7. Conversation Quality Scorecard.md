## 7. Conversation Quality Scorecard

Use this after each role-play (and after a real client conversation) to self-assess. Score each dimension 1–5, where 1 = “needs work” and 5 = “completely natural and effective.”

| Dimension | Score (1–5) | Notes / Evidence |
|-----------|:--:|------------------|
| **Redirect Effectiveness:** Did I successfully shift the conversation from hours to outcomes? Did the client acknowledge the outcome and articulate its value? | | |
| **Evidence Integration:** Did I naturally reference my pilot data, VRR numbers, outcome packaging clauses, or manifesto—without sounding like I was reading a script? | | |
| **Objection Handling:** Did I address objections without becoming defensive? Did I stay in the outcome frame rather than retreating to hourly justifications? | | |
| **Close Conviction:** Did the Future-Pacing Close feel like a natural conclusion, not a forced pitch? Did the client seem open to repeating the model? | | |

**Action steps:** For your lowest-scoring dimension, schedule an additional 15-minute focused drill. For example, if Evidence Integration scored low, practice interweaving your pilot data into the scripts until it feels organic.
