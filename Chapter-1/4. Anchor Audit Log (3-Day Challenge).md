## 4. Anchor Audit Log (3-Day Challenge)

For three consecutive working days, carry a small notebook or use your phone. Every time a thought crosses your mind that measures your worth in hours, utilization, or time spent, jot it down immediately. Then reframe it in outcome language. Use the table below as your template. Print three copies, one per day.

**Day:** ______________ **Date:** ______________

| Time | The Hour-Anchored Thought (what you caught yourself thinking) | The Outcome Reframe (rewrite the same situation through impact) |
|------|--------------------------------------------------------------|-----------------------------------------------------------------|
| | | |
| | | |
| | | |
| | | |
| | | |

*Examples to get you started:*  
- Hour-Anchored: “I only billed six hours today, I was so unproductive.”  
  Reframe: “Today I identified a compliance gap that, left alone, could have triggered a multi-million-dollar penalty. That single insight changed the client’s risk trajectory.”  
- Hour-Anchored: “That strategy session ran 90 minutes; the client won’t want to pay for that.”  
  Reframe: “In 90 minutes we reoriented the board’s regulatory approach, saving them months of wasted preparation and a probable enforcement action. The value of that meeting can’t be measured in minutes.”

At the end of three days, read through your reframes. Notice how your actual worth is far better captured by outcomes than by the clock.
