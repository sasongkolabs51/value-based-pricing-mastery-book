## 1. Pilot Readiness Pre‑Check  
*Before you choose a pilot, take a quick, honest reading of your current mindset and situation. No judgment—just a baseline.*

| Statement | 1 (Not at all) | 2 | 3 | 4 | 5 (Completely) |
|-----------|----------------|----|----|----|-----------------|
| I have a Chapter 4 heatmap with at least three clearly identified green‑ or yellow‑zone engagements. |  |  |  |  |  |
| I have run VRR calculations for those engagements and feel the numbers are defensible. |  |  |  |  |  |
| I have an outcome‑based engagement letter template (from Chapter 6) ready to adapt. |  |  |  |  |  |
| I can think of at least one client who trusts me deeply and might be open to a different fee structure. |  |  |  |  |  |
| I’m willing to have a conversation where I frame the new arrangement as a collaborative experiment, not a demand. |  |  |  |  |  |
| I accept that the pilot might surface things to fix—and that’s valuable, not a failure. |  |  |  |  |  |
| I’m committed to running only one or two small pilots before even thinking about scaling. |  |  |  |  |  |

**Interpreting your responses:**  
- If you scored low (1–2) on statements 1–3, revisit the earlier chapters and build those foundations first. A pilot without an audit, VRR, and packaging is a gamble.  
- Low scores on statements 4–5 are normal—this chapter exists to give you the safe, low‑risk first step.  
- Statement 7 is the most important: if you’re tempted to skip a pilot and go straight to scaling, re‑read the story of James. The pilot protocol is your insurance policy.
