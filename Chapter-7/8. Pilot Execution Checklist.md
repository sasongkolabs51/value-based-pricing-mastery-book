## 8. Pilot Execution Checklist

Follow this sequence from selection to debrief. Check off each item as you complete it.

**Phase 1: Pre-Flight**  
- [ ] Selected pilot candidate using 3-C Filter, with documented rationale.  
- [ ] Completed the One-Page Pilot Blueprint (Section 3).  
- [ ] Drafted pilot conversation script (Section 4).  
- [ ] Practiced the conversation aloud until it felt natural.  
- [ ] Ensured the engagement letter is ready (outcome statement, archetype, value assurance, no hourly language) per Chapter 6.

**Phase 2: The Conversation**  
- [ ] Scheduled a dedicated conversation with the client about the engagement structure.  
- [ ] Opened with a collaborative, experimental frame.  
- [ ] Presented outcome, fee, and assurance mechanism.  
- [ ] Listened more than talked; recorded client’s initial reaction.  
- [ ] Secured verbal agreement (written comes after, but aim for clear buy-in).  
- [ ] Documented any objections and how they were addressed in the Feedback Log.

**Phase 3: Execution**  
- [ ] Delivered work according to milestone plan.  
- [ ] Held all scheduled milestone check-ins, asking the pricing-model question each time.  
- [ ] Recorded feedback immediately after each check-in.  
- [ ] If scope change arose, paused, redefined outcome, and agreed on fee adjustment before continuing.  
- [ ] Delivered final outcome on or before agreed date.

**Phase 4: Post-Engagement**  
- [ ] Conducted formal debrief within one week of delivery, asking the three standard questions.  
- [ ] Completed Feedback Log with client verbatims.  
- [ ] Calculated pilot metrics (realization rate, satisfaction, willingness to repeat).  
- [ ] Compiled Pilot Results Summary.  
- [ ] Scheduled a personal reflection session: “What did I learn? What would I change?”  
- [ ] Identified the next pilot candidate based on learnings.
