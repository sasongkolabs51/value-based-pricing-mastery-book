## 5. Rapid Feedback Log Template  
*Purpose: Capture real‑time client reactions, questions, and objections so you can refine your approach and gather evidence for the internal sell.*

**Instructions:** Create a simple document (spreadsheet or notebook) with the columns below. Fill it in immediately after each interaction—phone call, email, milestone check‑in, informal chat. Verbatim quotes are gold; capture them exactly.

| Date | Interaction Type (e.g., kickoff call, milestone check‑in, email) | Client Contact | Key Quote / Question (verbatim if possible) | Your Interpretation / Learning | Action Taken or Planned |

|  |  |  |  |  |  |
|  |  |  |  |  |  |
|  |  |  |  |  |  |

**Special checkpoints to schedule:**  
- **Immediately after the initial conversation:** capture first reactions.  
- **At the midpoint (e.g., Day 45):** ask “How is the fixed‑fee arrangement feeling so far? Any concerns?”  
- **Within one week of final delivery:** conduct the formal debrief (see Section 6).
