## 1. Identity Alignment Pre‑Check  
*Before rebuilding, take stock of where your identity currently sits. Answer honestly—your goal is awareness, not a score.*

| Statement | Strongly Disagree (1) | Disagree (2) | Neutral (3) | Agree (4) | Strongly Agree (5) |
|-----------|----------------------|--------------|-------------|-----------|---------------------|
| When I introduce myself professionally, I describe the outcomes I create for clients, not the services I perform. |  |  |  |  |  |
| My sense of a “good week” is tied more to the client problems I solved than to the hours I logged. |  |  |  |  |  |
| I can name—off the top of my head—the three specific, measurable results that only I seem to deliver across different clients. |  |  |  |  |  |

| The idea of never billing by the hour again feels exciting and aligned with my worth, not terrifying. |  |  |  |  |  |

**Interpreting your responses:**  
- If you scored mostly 1s and 2s, the old “trusted advisor” identity still holds sway. That’s not a failure—it’s just where many of us start. The exercises in this toolkit will feel particularly liberating for you.  
- If you’re in the 3–4 range, you’ve already begun the shift but haven’t fully embodied it. The drills will crystallize what you already sense.  
- If you marked mostly 5s, this chapter will still give you sharper language and a more systematic approach to something you’re already living.

Return to this check after completing the toolkit. Notice the distance you’ve traveled.
