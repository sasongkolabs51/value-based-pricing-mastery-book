*Before you redesign any workflow, take a quick, honest reading of where your internal model currently stands.*

| Statement | 1 (Strongly Disagree) | 2 | 3 | 4 | 5 (Strongly Agree) |
|-----------|----------------------|----|----|----|---------------------|
| My team’s compensation and performance evaluations are currently based primarily on billable hours or utilization. |  |  |  |  |  |
| I have seen talented senior associates or managers resist using AI tools because they fear it will reduce their billable hours or bonuses. |  |  |  |  |  |
| I can name at least two people on my team whose real value lies in judgment and client trust, not in hours logged. |  |  |  |  |  |
| I have a clear idea of which parts of my current workflow could be done faster (or are already being done) by AI. |  |  |  |  |  |
| I feel confident I could redesign compensation for at least one person on my team without waiting for firm‑wide policy changes. |  |  |  |  |  |
| My current quality assurance process relies more on “review by a senior person” than on documented checkpoints. |  |  |  |  |  |
| I’ve run the numbers on at least one engagement to compare old‑model margin vs. new‑model margin when delivery is AI‑augmented. |  |  |  |  |  |

**Interpreting your responses:**  
- High agreement with statements 1 and 2 signals the broken leverage model is actively hurting your practice. The compensation pilot (Section 5) is your most urgent tool.  
- If you scored low on statements 3–5, you may need to have some honest conversations with your team about where value really lives. The role‑mapping exercise (Section 2) will help.  
- Statement 7 is the financial heart of the chapter. If you haven’t run the numbers yet, the Margin Simulation (Section 7) will be an eye‑opener.
