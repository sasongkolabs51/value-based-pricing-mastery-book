**Instructions for creating the fillable Word file:**

1. Copy the above content into a blank Word document.
2. Enable the **Developer** tab (File → Options → Customize Ribbon → check Developer).
3. Replace each bracketed instruction like `[Insert Number field]` with a **Rich Text Content Control** or **Text Form Field** from the Developer tab, leaving the placeholder text inside. This makes it fill‑in‑the‑blank while keeping the template structure intact.
4. Protect the document (Developer tab → Restrict Editing → allow only filling in forms) so users can tab through fields without accidentally altering the fixed text.
5. Save as `.docx`.
Protect the document so only the form fields are editable, giving a clean, fill‑in‑the‑blank experience.
